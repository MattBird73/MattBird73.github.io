Capture The Flag AI

Two teams of AI battle it out in a game of capture the flag against one another. These AI are built using a mix of decision trees and state machines to give them realistic and human like behaviors allowing for an immersive and challenging single player experience.

The idea for this project was that the AI could be used for any sort of combat game, either to fight against or alongside a player. The AI are able to make informed desicions based on the environment around them. Their capabilities include deciding when to attack enemies, when to try and collect the enemy flag, when to defend their own flag and when to pick up and utilise power-ups and health kits.

Behind the decision trees

This abstract base class is a template for each node on the tree. It stores information regarding the type of node (leaf/decision) and either performs an action or makes a decision depending on its type.

The action and decision nodes are both subclasses of Node. The action node stores a reference to the AI agent and the action that needs to be executed when the node is reached. The decision node stores references two ‘child’ nodes which it will decide between when it is called. Each node stores a reference to the node that comes after it in the tree, that way when the current node has been evaluated the tree moves on to the next node in the sequence straight away.

The actual tree works by assinging a root node as a place to start, then the tree is traversed each update until a leaf node is reached. The leaf node will trigger an action, then the process starts again.
